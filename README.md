# Browser-UI
ブラウザ用UI

## 開発動機
[shadcn/ui](https://github.com/shadcn-ui/ui)に衝撃を受けて自分でもできるか挑戦したいというモチベーションからスタートしています

## 開発方針
- 基本的にはコンポーネント用のクラスを使用して記述する
- calc等の計算が必要な場合は各クラスを用意するのは不可能なのでstyleに直接記述する
- hover等が必要な場合にはCSSに記述する
- ショートハンドは使用しない
